The folder Graph_gen_ReLU_network contains the graphs generated by the proposed ReLU networks.
The folder Training_data_grnn_gdp contains the training graphs used for GraphRNN and GraphGDP models. These graphs are the same as the graphs in the folder Graph_gen_ReLU_network.
The folder Graph_gen_grnn contains the graphs generated by the GraphRNN model.
The folder Graph_gen_gdp contains the graphs generated by the GraphGDP model. 
The folder Graph_dist contains the files with the graph edit distance of the original graph from the graphs generated by the proposed ReLU networks, GraphRNN, and GraphGDP models.
